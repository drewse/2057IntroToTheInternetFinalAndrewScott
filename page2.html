Web Technologies - Understanding Protocols and Standards

Table 1

Protocol Description
HTTP Hypertext Transfer Protocol - the basis for data transmission on the Internet.
HTML Hypertext Markup Language - common markup language used in web apps and pages.

Table 2

Protocol Description
FTP File Transfer Protocol - common network protocol used on computer networks to transmit files between clients and servers.
SMTP Simple Mail Transfer Protocol - protocol standard for sending electronic mail (email) via Internet Protocol (IP) networks.
